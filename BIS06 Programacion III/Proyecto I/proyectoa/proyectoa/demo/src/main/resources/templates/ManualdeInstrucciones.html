Bienvenido al Block de Notas

Tiene como objetivo ser una guía para el uso y la interacción dentro del mismo, donde puede editar, guardar, imprimir y dar un formato determinado. Es una forma sencilla de guardar información importante para el usuario.

Pasos para usar el Block de Notas:

  1. Debe ingresar al sitio web.
  2. Si el usuario no tiene cuenta, debe crear una.
  3. Para crear una cuenta, debe tener un usuario con un correo electrónico y contraseña.
  4. Una vez que tiene usuario, debe ingresar a la dirección y colocar su usuario y contraseña.
  5. Dentro de la página puede hacer lo siguiente:
    • Crear una nota.
    • Escribir el texto deseado.
    • Guardarlo.
    • Puede editarlo.
    • En cada sesión debe guardar los cambios.
  6. Cada vez que ingrese al sitio web, debe colocar usuario y contraseña.
  7. Cada nota se guarda en la base de datos.
  8. Cada vez que se cierra la sesión, se debe asegurar que se haya guardado la información.
  9. Una vez guardada la información, se puede cerrar la sesión de forma segura con los datos elaborados.
